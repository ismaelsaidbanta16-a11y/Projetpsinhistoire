Introduction au CSS

Apprendre à styliser une page web

1. Qu'est-ce que le CSS ?

Le CSS est un langage utilisé pour définir le style et la présentation des pages web créées en HTML.

Il permet de contrôler les couleurs, les polices, les marges et la disposition des éléments sur une page.

Grâce au CSS, on peut transformer une page HTML simple en un site moderne et attractif.

Le CSS fonctionne en appliquant des règles aux éléments HTML à l’aide de sélecteurs comme les balises, classes ou identifiants.

2. Sélecteurs

On peut cibler des éléments HTML :

  • par balise (ex: h1)
  • par classe (ex: .highlight)
  • par identifiant (ex: #box)
